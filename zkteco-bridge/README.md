# ZKTeco Firebase Bridge - كمبيوتر العمل

## كيف يعمل

```
[أنت في البيت - اضغط زر] → [Firebase] ← [هذا البرنامج في العمل] → [جهاز البصمة]
```

## التثبيت (مرة واحدة على كمبيوتر العمل)

```
cd zkteco-bridge
npm install
```

## التشغيل

```
node bridge.js
```

على ويندوز (الأسهل):

شغّل الملف:

start-bridge.cmd

هذا الملف يقوم بالتثبيت تلقائياً لأول مرة ثم يشغّل الـ Bridge.

أو مع تحديد IP جهاز البصمة مباشرة:

```
ZK_IP=192.168.1.100 ZK_PORT=4370 node bridge.js
```

## الإعداد

افتح `bridge.js` وغيّر:
```js
const ZK_IP   = '192.168.1.100';  // IP جهاز البصمة في العمل
const ZK_PORT = 4370;
```

## بعد التشغيل

اضغط الزر من موقعك في البيت → سيستلمه البرنامج في العمل خلال ثوانٍ → تظهر البيانات في الموقع.
